# Frontend UI Component Lab

A curated collection of high-fidelity UI components built with a focus on CSS architecture and responsive design. This repository documents the implementation of design mockups into functional, mobile-first web components using Semantic HTML5 and modern CSS3.

##  Live Components

### 1. [View Order Summary Card](https://rijul2006.github.io/Frontend-UI-Components/order-summary-card/) 
A multi-section UI component featuring a graphical header, descriptive body text, and interactive action buttons.
- **Key Features:** Custom CSS Box-Shadows and hover states for visual depth.

### 2. [View QR Code Component](https://rijul2006.github.io/Frontend-UI-Components//qr-code-component/)
A pixel-perfect, centered card layout designed for mobile-first accessibility.
- **Key Features:** Precise vertical and horizontal alignment using Flexbox.

##  Technical Implementation
- **Responsive Design:** Optimized for all device viewports using CSS Media Queries.
- **Modern CSS:** Heavy use of Flexbox for layout management.
- **Academic Context:** Developed by a University Rank 2 student (9.8 CGPA) focusing on engineering precision.
